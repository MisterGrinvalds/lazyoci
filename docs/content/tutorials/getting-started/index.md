# Getting Started with lazyoci

In this tutorial series, we'll get you up and running with lazyoci. By the end, you'll know how to install the tool, browse container registries, and pull your first artifact.

## What We'll Accomplish

1. **Install lazyoci** on your system
2. **Launch the TUI** and explore the interface
3. **Browse Docker Hub** to find container images
4. **Pull an image** both through the TUI and CLI

## What You'll See

- The lazyoci terminal user interface in action
- How to navigate between different registries
- Detailed information about container images and their layers
- Your first successful artifact pull

## Time Required

About 10-15 minutes total, broken into short steps.

## Prerequisites

- A terminal or command prompt
- Internet connection to access public registries
