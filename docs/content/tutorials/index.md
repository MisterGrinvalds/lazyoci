# Tutorials

Welcome to the lazyoci tutorials! These step-by-step guides will help you learn lazyoci by doing. Each tutorial is hands-on and designed to produce immediate, visible results.

## Getting Started Series

Perfect for your first time using lazyoci. We'll install the tool, browse Docker Hub, and pull your first container image.

### [Getting Started with lazyoci](./getting-started/)
- **What you'll learn**: Install lazyoci, navigate registries, and pull artifacts
- **Time estimate**: 10-15 minutes
- **Prerequisites**: None

## Working with Private Registries

Learn how to connect lazyoci to your organization's private container registries.

### [Private Registries](./private-registries/)
- **What you'll learn**: Add authentication, browse private repos, and use Docker credentials
- **Time estimate**: 10 minutes
- **Prerequisites**: Access to a private registry

## Local Development Setup

Set up a local development environment with a test registry for experimenting with different artifact types.

### [Local Development](./local-dev/)
- **What you'll learn**: Run a local registry, push test artifacts, and explore artifact types
- **Time estimate**: 15 minutes
- **Prerequisites**: Docker installed

## Building and Pushing Artifacts

Learn how to build container images, Helm charts, and generic OCI artifacts using the `.lazy` config format and push them to registries.

### [Building Artifacts](./building/)
- **What you'll learn**: Create `.lazy` configs, build different artifact types, use tag templates, and push to registries
- **Time estimate**: 20 minutes
- **Prerequisites**: Docker installed, a registry to push to (local or remote)

---

Each tutorial follows a learning-oriented approach where we work together to accomplish real tasks. Let's get started!
